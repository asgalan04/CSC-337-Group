Plane Ticket Booking System

This is a Node.js web application built using Express and MongoDB.  
Follow the steps below to install dependencies and run the project.

1. Install required packages:
    npm install express
    npm install mongodb

2. Start the MongoDB server:
    mongod

3. Start the web server:
    node server.js

The application will run at http://localhost:8080


MongoDB Collections Used

1. Destinations Module:
   use flightDB
   db.destinations.find()

2. Flight Booking Module:
   use buyingDB
   db.newCollection.find()
